Description

Kenpo Flashcards (Web) is a lightweight web app + server that helps you study Kenpo vocabulary, track learning status, and sync progress and breakdowns across devices.

Major features

  • Study tabs: Unlearned, Unsure, Learned, All
  • Group/Subgroup filtering + “All Cards” mode
  • Search, counters, and fast browsing
  • Cross-device sync for progress and breakdowns
  • AI breakdown autofill (optional)
