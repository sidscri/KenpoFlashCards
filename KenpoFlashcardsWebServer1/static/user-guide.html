Overview

Kenpo Flashcards helps you study vocabulary and track progress across devices. Cards are organized by group/subgroup and you can move a term between Unlearned, Unsure, and Learned. The web server can also store and sync AI-powered breakdowns.

Sync (Android ↔ Web)

  • Progress sync: push/pull learning status.
  • Breakdown sync: saved breakdowns can be pulled on other devices.
  • Card IDs are normalized using the server helper mapping for consistent cross-device matching.
